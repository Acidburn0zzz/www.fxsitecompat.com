非標準メソッド [`window.openDialog`](https://developer.mozilla.org/ja/docs/Web/API/window.openDialog) が Web コンテンツから使用できなくなりました。代わりに [`window.open`](https://developer.mozilla.org/ja/docs/Web/API/window.open) を使用してください。
