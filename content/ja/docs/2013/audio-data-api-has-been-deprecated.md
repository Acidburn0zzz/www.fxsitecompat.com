非標準で実験的な [Audio Data API](https://developer.mozilla.org/ja/docs/Introducing_the_Audio_API_Extension) が、非推奨、廃止予定となりました。代わりに標準化された [Web Audio API](https://developer.mozilla.org/ja/docs/Web_Audio_API) を使用してください。
